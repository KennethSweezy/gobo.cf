## What is Gobo.cf?
Gobo.cf is a URL Shortener For Scratch Projects started by @Thermodynamical

## Shortened URLs

| URL           | Function           |
| ------------- | ------------------ |
| /u/?to=       | To User Page       |
| /s/?to=       | To Studio Page     |
| /p/?to=       | To Project Page    |
| /p/e/?to=     | To Project Editor  |
| /d/?to=       | To Topic           |
